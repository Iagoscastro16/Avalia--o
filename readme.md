### Aether Labs - Site institucional
Site institucional estático desenvolvido como projeto de avaliação, com múltiplas páginas, responsividade e boas práticas de acessibilidade.

### Páginas 

-**Início**(`index.html`)- hero section com apresentação e cards diferenciais.

-**Sobre(`Sobre.html`)**- Informações sobre a empresa

-**Serviços(`servicos.html`)**- serviços oferecidos

-**Contato(`contato.html`)**- formulário de contato

-**Obrigado(`obrigado.html`)**- confirmação de envio

### Estrutura

```
assets/
├── base.css        # Reset, header, footer, layout base
├── style.css       # Hero, cards, grid de destaques
├── contato.css     # Estilos da página de contato
├── servicos.css    # Estilos da página de serviços
├── sobre.css       # Estilos da página sobre
└── obrigado.css    # Estilos da página de confirmação
Index.html
contato.html
servicos.html
sobre.html
obrigado.html
```

### Funcionalidades 

- Layout reponsivo com CSS Grid e Flexbox

- Menu hambúguer para mobile(CSS puro, sem javascript)

- Cards de acessibilidade com grid responsivo

- Acessibilidade com `arial-label` e `aria-labelledby`

- Ano do footer gerado dinamicamente via JavaScript

- CSS separado por página para melhor organização

### Como rodar
 
Abra o arquivo `Index.html` diretamente no navegador — sem dependências ou instalação, ou use o live server no vs code.
